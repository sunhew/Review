0-1. 자료형

자료형(Data Type)은 프로그래밍에서 데이터의 종류를 나타내는 것으로, 다음과 같은 종류가 있습니다.

1. typeof 연산자: 변수의 자료형을 확인할 수 있습니다.
2. instanceof 연산자: 객체의 생성자 함수를 확인할 수 있습니다.
3. Object.prototype.toString.call(): 객체의 자료형을 문자열로 반환합니다. 

 {
                        var num = 10;
                        console.log(typeof num);  //typeof 연산자

                        var arr = [1, 2, 3];
                        console.log(arr instanceof Array); //instanceof 연산자

                        var str = "Hello";
                        console.log(Object.prototype.toString.call(str)); //Object.prototype.toString.call()
}

1.정수형(Integer): 소수점이 없는 숫자를 저장하는 자료형입니다. 예를 들어, 10, -5 등이 정수형에 해당합니다.

2.실수형(Float): 소수점이 있는 숫자를 저장하는 자료형입니다. 예를 들어, 3.14, -2.5 등이 실수형에 해당합니다.

3.문자형(Character): 문자를 저장하는 자료형입니다. 예를 들어, 'A', 'B', 'C' 등이 문자형에 해당합니다.

4.문자열형(String): 여러 개의 문자를 묶어서 저장하는 자료형입니다. 예를 들어, "Hello", "World" 등이 문자열형에 해당합니다.

5.불리언형(Boolean): 참(True) 또는 거짓(False)을 저장하는 자료형입니다. 예를 들어, if문이나 while문 등에서 조건을 판단할 때 사용됩니다.

6.객체형(Object): 여러 개의 변수와 함수를 묶어서 저장하는 자료형입니다. 예를 들어, class Person { String name; int age; } 과 같이 선언하여 사용할 수 있습니다.
위와 같은 자료형은 프로그래밍 언어마다 조금씩 다를 수 있습니다. 예를 들어, C언어에서는 int, char, float 등의 자료형을 사용하고, Java에서는 int, char, String 등의 자료형을 사용합니다.

0-2. 연산자

연산자(operator) 는 프로그래밍에서 데이터를 조작하고 계산을 수행하는 데 사용되는 기호입니다. 다음은 대표적인 연산자들입니다.

1.산술 연산자: 숫자를 계산하는 데 사용되는 연산자입니다.
2.대입 연산자: 변수에 값을 할당하는 데 사용되는 연산자입니다.
3.비교 연산자: 두 값을 비교하여 참 또는 거짓을 반환하는 연산자입니다.
4.논리 연산자: 논리적인 조건을 처리하는 데 사용되는 연산자입니다.
5.비트 연산자: 비트 단위로 연산을 수행하는 데 사용되는 연산자입니다.
6.삼항 연산자: 조건에 따라 다른 값을 반환하는 데 사용되는 연산자입니다. 

 {
                        let a = 10;
                        let b = 20;

                        console.log(a + b); // 30
                        console.log(a - b); // -10
                        console.log(a * b); // 200
                        console.log(a / b); // 0.5
                        console.log(a % b); // 10       //산술 연산자      
                        
                        let a = 10;

                        a += 20; // a = a + 20 => a는 이제 30
                        a -= 5; // a = a - 5 => a는 이제 25
                        a *= 2; // a = a * 2 => a는 이제 50
                        a /= 10; // a = a / 10 => a는 이제 5  //할당 연산자

                        let a = 10;
                        let b = 20;

                        console.log(a == b);  // false
                        console.log(a != b);  // true
                        console.log(a > b);   // false
                        console.log(a < b);   // true
                        console.log(a >= b);  // false
                        console.log(a <= b);  // true           //비교 연산자

                        let a = true;
                        let b =;

                        console.log && b  //
                        console(a || b  true
                        console.loga);      // false            //논리 연산자

                        let age = 15;

                        let beverage = (age >= 21) ? "Beer" : "Juice";
                        
                        console.log(beverage);  // 출력: Juice      //삼항 연산자
}

1.산술 연산자: 이들은 수학적인 계산을 수행하는데 사용됩니다.
예시: + (덧셈), - (뺄셈), * (곱셈), / (나눗셈), % (나머지), ++ (1 증가), -- (감소) 등

2.할당 연산자: 이들은 변수에 값을 할당하는데 사용됩니다.
예시: = (할당), += (덧셈 후 할당), -= (뺄셈 후 할당), *= (곱셈 후 할당), /= (나눗셈 후 할당) 등

3.비교 연산자: 이들은 두 값의 비교를 위해 사용됩니다.
예시: == (동등), != (부등), > (초과), < (미만),>= (이상), <= (이하),===(일치), !==(불일치) 등

4.논리 연산자: 이들은 논리적인 연산을 수행하는데 사용됩니다.
예시: && (논리곱), || (논리합), ! (부정)

5.대입 연산자: 변수에 값을 할당하는 데 사용되는 연산자입니다.
=: 할당, +=: 더하기 후 할당, -=: 빼기 후 할당, *=: 곱하기 후 할당, /=: 나누기 후 할당

6.삼항 연산자: 이 연산자는 세 개의 피연산자를 가지며, 조건에 따라 다른 결과를 반환합니다.
이름에서 알 수 있듯이 세 가지 부분으로 구성되어 있습니다: 조건, 참일 때의 값, 그리고 거짓일 때의 값입니다.
(예시: 조건 ? 값1 : 값2) 조건이 참이면 값1이 반환되고, 거짓이면 값2가 반환됩니다. 위의 예제에서는 age가 21 이상인지 확인하는 조건을 삽입했습니다.
이 조건이 참이면 "Beer"를 반환하고, 그렇지 않으면 "Juice"를 반환합니다.
이 경우 age는 15이므로 조건은 거짓이며, 따라서 "Juice"가 반환됩니다.
삼항 연산자는 간단한 if-else 문을 한 줄로 줄일 수 있어 코드를 간결하게 만드는 데 유용합니다.

복잡한 로직에서는 가독성을 해칠 수 있으므로 주의해서 사용해야 합니다.

02. 변수 : 데이터 저장 + 데이터 변경

위의 코드는 자바스크립트에서 변수를 선언하고, 값을 할당하고, 값을 변경한 후, 변경된 값을 출력하는 예시입니다.

}

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

이 코드에서는 변수에 저장된 값에 다른 값을 더하거나 연결하여 데이터를 추가합니다.

x += 200;은 x에 저장된 값에 200을 더한 결과를 다시 x에 저장하는 것입니다.
y += 300;은 y에 저장된 값에 300을 더한 결과를 다시 y에 저장하는 것입니다.
z += "jquery";는 z에 저장된 문자열 뒤에 "jquery"를 연결한 결과를 다시 z에 저장하는 것입니다. 

 {
        let x = 100;
        let y = 200;
        let z = "javascript";

        x += 200;   //x = x + 200 의 약자
        y += 300;
        z += "jquery";

        console.log(x);
        console.log(y);
        console.log(z);
}

이 방법은 데이터를 추가하거나 수정할 때 자주 사용되는 방법 중 하나입니다.

04. 상수 : 데이터 저장 + 데이터 변경(x)

변수 코드와 비슷하지만, const 키워드를 사용하여 변수를 선언하고 있습니다.
const 키워드는 변수의 값을 한 번만 할당할 수 있으며, 이후에는 값을 변경할 수 없습니다.
따라서, 위의 코드에서 x 변수에 300을 할당하려고 하면 오류가 발생합니다.
코드를 실행하면 다음과 같은 오류가 발생합니다. 

 {
        const x = 100;
        const y = 200;
        const z = "javascript";

        x = 300;

        console.log(x);
        console.log(y);
        console.log(z);
}

const 키워드를 사용하여 변수를 선언할 때는 값을 한 번만 할당하고, 이후에는 값을 변경하지 않도록 주의해야 합니다.

09. 배열 : 데이터 저장(여러개) : 표현방법5 : 2차 배열

배열 arr의 각 요소에 접근하여 값을 출력하는 코드입니다. 

 {
        const arr = [100, [200, 300], ["javascript", "jquery"]];

        console.log(arr[0]);    //100
        //console.log(arr[1][0]);    //200
        console.log(arr[1][1]);    //300
        console.log(arr[2][0]);   //javascript
        console.log(arr[2][1]);  //jquery
        //console.log(arr[1]);     //200, 300
}

arr[0]은 배열의 첫 번째 요소인 100을 출력합니다. arr[1][0]은 배열의 두 번째 요소인 배열 [200, 300]의 첫 번째 요소인 200을 출력합니다.
