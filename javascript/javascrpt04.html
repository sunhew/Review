01. if문(조건문)

if 조건문은 프로그래밍에서 조건에 따라 다른 코드를 실행하기 위해 사용되는 구문입니다.
조건문은 조건식을 평가하여 그 결과에 따라 다른 코드를 실행합니다. 

 {
         true : true, "문자열", 1, 2, "1", "2"
         false : false, 0, null, undefined, ""(빈 문자열)
         if (0) {
             document.write("실행되었습니다.(true)")
        } else {
            document.write("실행되었습니다.(false)")
        }
}

해당 코드는 if 조건문을 사용하여 조건에 따라 다른 코드를 실행하는 예시입니다.
객체는 중괄호({})로 둘러싸이며, 속성과 값이 콜론(:)으로 구분되어 나열됩니다.
if문은 조건식을 평가하여 조건이 true인 경우 if문 내부의 코드를 실행하고, false인 경우 else문 내부의 코드를 실행합니다.
위 코드에서는 조건식으로 0을 사용했는데, 0은 false로 평가되기 때문에 else문 내부의 코드가 실행되어 "실행되었습니다.(false)"라는 문자열이 출력됩니다.
조건식에는 true, false, 숫자, 문자열, 객체 등 다양한 값을 사용할 수 있습니다.
조건식의 평가 결과가 true인 경우 if문 내부의 코드를 실행하고, false인 경우 else문 내부의 코드를 실행합니다.
if문은 조건에 따라 다른 코드를 실행해야 하는 경우에 유용하게 사용됩니다.

if 조건문 키워드

else: 조건식이 거짓일 때 실행할 코드를 정의합니다.

elif: 여러 개의 조건 중 하나 이상이 참일 때 실행할 코드를 정의합니다.

02. 다중 if문

이 코드는 다중 if문을 사용하여 변수 num의 값에 따라 다른 동작을 수행합니다.

 {
        const num = 100;
        if (num == 90) {
            document.write("실행되었습니다. (num = 90)");
        } else if (num == 95) {
            document.write("실행되었습니다. (num = 95)");
        } else if (num == 100) {
            document.write("실행되었습니다. (num = 100)");
        } else if (num == 105) {
            document.write("실행되었습니다. (num = 105)");
        }
}

1. 첫 번째 if문은 num이 90인지 확인하고, 만약 그렇다면 "실행되었습니다. (num = 90)"이라는 메시지를 출력합니다.
2. 두 번째 if문은 num이 95인지 확인하고, 만약 그렇다면 "실행되었습니다. (num = 95)"이라는 메시지를 출력합니다
3. 세 번째 if문은 num이 100인지 확인하고, 만약 그렇다면 "실행되었습니다. (num = 100)"이라는 메시지를 출력합니다.
4. 네 번째 if문은 num이 105인지 확인하고, 만약 그렇다면 "실행되었습니다. (num = 105)"이라는 메시지를 출력합니다.

만약 num의 값이 위의 조건 중 어느 것에도 해당하지 않는다면, else문이 실행되어 "실행되지 않았습니다."라는 메시지가 출력됩니다.

다중 if문 키워드
다중 if문은 여러 조건을 순차적으로 검사하여 해당하는 조건에 따라 다른 동작을 수행하는 데 사용됩니다. 이를 통해 코드의 가독성과 유연성을 높일 수 있습니다.

03. 중첩 if문

중첩 IF문은 IF문 안에 또 다른 IF문이 포함되어 있는 형태입니다
아래 코드는 중첩된 if문을 사용하여 조건을 검사하고, 그에 따라 문서에 특정 텍스트를 출력하는 예제이며
아래 코드에는 num이 100인지 확인하는 첫 번째 IF문이 있고, 그 안에 num이 100인지 다시 확인하는 두 번째 IF문이 있습니다. 

 {
        const num = 1
        if (num === 100) {
            document.write("실행되었습니다. (num = 1)");
            if (num === 100) {
                document.write("실행되었습니다. (num = 2)");
            }
        } else {
            document.write("실행되지 않았습니다. (num = 4)");
        }
}

1.첫 번째 if문은 변수 num이 100과 같은지 검사합니다. 만약 같다면, "실행되었습니다. (num = 1)"이라는 텍스트를 문서에 출력합니다.
객체는 중괄호({})로 둘러싸이며, 속성과 값이 콜론(:)으로 구분되어 나열됩니다.

2.첫 번째 if문 내부에 또 다른 if문이 있습니다. 이 if문은 변수 num이 100과 같은지 다시 검사합니다.
만약 같다면, "실행되었습니다. (num = 2)"라는 텍스트를 문서에 출력합니다.

3.첫 번째 if문이 false라면, else문이 실행되어 "실행되지 않았습니다. (num = 4)"라는 텍스트를 문서에 출력합니다.

중첩 if문 키워드
중첩된 if문은 여러 조건을 검사해야 할 때 유용하게 사용됩니다.
하지만, 너무 많은 중첩은 코드를 복잡하게 만들 수 있으므로, 필요한 경우에만 사용하는 것이 좋습니다.

04. if문 생략 & 삼항연산자(조건식 연산자)

04-1은 if문 생략을 다루고 있습니다.

Python에서는 조건문을 간결하게 작성하기 위해 if문을 생략할 수 있습니다. 이는 조건문이 참일 때만 코드를 실행하는 경우에 유용합니다. 

 {
        const num = 100;
        if (num == 100) {
            document.write("실행되었습니다. (true)")
        }
        if (num == 100) document.write("실행되었습니다.(true)")
        if (num == 100) {
            document.write("실행되었습니다. (true)")
        } else {
            document.write("실행되었습니다. (false)")
        }
        if (num == 100) document.write("실행되었습니다.(true)")
        else document.write("실행되었습니다.(false)")
}

1.조건문이 참일 때만 실행하는 코드를 콜론(:)으로 감싸기
(예시)
x = 10
print(x) if x > 0 else None
위 코드에서는 x가 0보다 큰 경우에만 print 함수를 실행합니다. 만약 x가 0보다 작다면, print 함수는 실행되지 않습니다.

2.조건문이 참일 때 실행하는 코드와 거짓일 때 실행하는 코드를 함께 작성하기
(예시)
x = 10
print(x) if x > 0 else print("x는 0보다 작습니다.")
위 코드에서는 x가 0보다 큰 경우에는 print 함수를 실행하고, 그렇지 않은 경우에는 "x는 0보다 작습니다."라는 메시지를 출력합니다.

if문 생략하기 키워드
이러한 방법으로 if문을 생략하면 코드를 간결하게 작성할 수 있으며, 조건문이 참일 때만 코드를 실행하는 경우에 유용합니다.
하지만, if문을 생략하는 것이 항상 좋은 것은 아니며, 상황에 따라 적절한 방법을 선택해야 합니다.

04. if문 생략 & 삼항연산자(조건식 연산자)

04-1은 삼항 연산자를 다루고 있습니다.

삼항 연산자는 조건문을 간단하게 작성할 수 있는 연산자입니다.
해당 코드는 num 변수가 100과 같은지 비교하고, 같다면 "true"를 출력하고, 그렇지 않다면 "false"를 출력합니다. 

 {

(예시)
조건문 ? 참일 때 실행할 코드 : 거짓일 때 실행할 코드;

위의 코드를 예로 들면, num이 100과 같은지 비교한 후, 조건이 참이라면 "true"를 출력하고, 거짓이라면 "false"를 출력합니다.

삼항 연산자 키워드
삼항 연산자는 조건문을 간단하게 작성할 수 있기 때문에, 자주 사용되는 연산자 중 하나입니다.
하지만, 삼항 연산자를 과도하게 사용하면 코드가 복잡해질 수 있으므로, 적절한 곳에 사용하는 것이 중요합니다.

05. switch문

switch문은 주어진 조건에 따라 여러 가지 경우를 처리할 수 있는 제어문입니다.
조건문은 조건식을 평가하여 그 결과에 따라 다른 코드를 실행합니다. 

 {
        const num = 1
        switch (num) {
            case 90:
                document.write("실행되었습니다.(90)");
                break;
            case 100:
                document.write("실행되었습니다.(100)");
                break;
            case 105:
                document.write("실행되었습니다.(105)");
                break;
            case 110:
                document.write("실행되었습니다.(110)");
                break;
            default:
                document.write("실행되었습니다.");
        }

        (ex)
        switch (조건) {
            case 값1:
                // 값1에 해당하는 경우 실행할 코드
                break;
            case 값2:
                // 값2에 해당하는 경우 실행할 코드
                break;
            default:
                // 위의 case에 해당하지 않는 경우 실행할 코드
        }
}

위의 예시에서는 변수 num의 값이 90, 100, 105, 110 중 하나인 경우 해당 case의 코드를 실행하고, 그 외의 값인 경우 default case의 코드를 실행합니다.
각 case의 코드는 break 문을 사용하여 종료됩니다.
만약 break 문이 없다면, 해당 case의 코드가 실행된 후 다음 case의 코드도 이어서 실행됩니다.
이와 같이 switch문은 조건에 따라 다양한 동작을 수행할 수 있어 코드의 가독성을 높이고, 반복문을 사용하지 않고도 여러 가지 경우를 처리할 수 있습니다.

07 do while문 (반복문)

해당 코드는 1부터 10까지의 숫자를 출력하는 do-while 반복문입니다.
do-while 반복문은 조건을 먼저 확인하는 다른 반복문들과는 달리, 먼저 코드를 실행한 후 조건을 확인합니다.
따라서 조건이 거짓이더라도 최소 한 번은 코드를 실행합니다. 

 {
        let num = 1;
        do {
            document.write(num);
            num++;
        } while (num <= 10);
}

위 코드에서는 변수 num을 1로 초기화하고, do-while 반복문을 시작합니다.
반복문 내부에서는 document.write() 함수를 사용하여 현재 num의 값을 출력하고, num을 1씩 증가시킵니다.
이후 num이 10보다 작거나 같은지 확인합니다. 만약 num이 10보다 작다면, 다시 반복문을 실행합니다.
이 과정을 num이 10이 될 때까지 반복합니다.

08 for 문 (반복문)

for문은 초기값, 조건식, 증감식으로 구성되며, 조건식이 참인 동안 반복문 내의 코드를 실행합니다.
조건문은 조건식을 평가하여 그 결과에 따라 다른 코드를 실행합니다. 

 {
        for (let i = 1; i <= 10; i++) {
            console.log(i);
        }
}

for (let i = 1; i <= 10; i++) : 이 부분은 for 반복문의 초기화, 조건식, 증감식을 나타냅니다.

초기화 : let i = 1은 변수 i를 1로 초기화합니다.

조건식 : i <= 10은 i가 10 이하일 때 반복문을 실행합니다.

>

증감식 : i++은 반복문이 실행될 때마다 i의 값을 1씩 증가시킵니다.

>

document.write(i); : 이 부분은 반복문이 실행될 때마다 변수 i의 값을 출력합니다.

>

정답 확인하기
이 코드를 실행하면 1부터 10까지의 숫자가 출력됩니다.

09 break문

break문은 반복문이나 switch문에서 사용되며, 현재 실행 중인 반복문이나 switch문을 즉시 종료하는 역할을 합니다. 

 {
        for (let i = 1; i <= 10; i++) {
            if (i == 5) {
                break;
            }
            document.write(i, "
"); //1 2 3 4
        }
}

위의 예시 코드에서는 for문을 사용하여 1부터 10까지의 숫자를 출력하는 반복문을 실행하고 있습니다만,
반복문 내부에 if문을 사용하여 i가 5일 때 break문을 실행하도록 설정되어 있습니다.
따라서 i가 5일 때 break문이 실행되어 반복문이 즉시 종료되고, 그 이후의 코드는 실행되지 않습니다.
그 결과로 출력되는 값은 1, 2, 3, 4 입니다.
이처럼 break문은 특정 조건에서 반복문을 종료해야 할 때 유용하게 사용됩니다.

정답 확인하기
1, 2, 3, 4

10. continue

이 코드는 for 반복문과 continue 문을 함께 사용하는 좋은 예입니다.
조건문은 조건식을 평가하여 그 결과에 따라 다른 코드를 실행합니다. 

 {

        for (let i = 1; i <= 10; i++) {
            if (i == 5) {
                continue;
            }
            document.write(i, "
");
        }
}

반복문 for (let i = 1; i <= 10; i++)는 i가 1부터 시작해서 10이 될 때까지 반복하도록 설정되어 있습니다.
즉, 이 코드는 i 값이 1부터 10까지 변하는 동안 코드 블록 내의 명령을 반복 실행합니다.

반복 내부에서는 if (i == 5) 조건문이 있습니다. 이 조건문은 i의 값이 5인 경우에만 실행됩니다.
즉, i가 5일 때, continue; 문이 실행되고, 이는 현재 반복을 중단하고 바로 다음 반복으로 넘어가라는 명령입니다.
따라서 i가 5일 때는 document.write(i, "
"); 코드가 실행되지 않습니다.

마지막으로, document.write(i, "
"); 코드는 i의 값을 문서에 출력하고, 그 다음 줄로 넘어가는 명령이며 "
"는 HTML에서 줄 바꿈을 의미합니다.
결과적으로, 이 코드는 1부터 10까지의 숫자를 출력하는데, i가 5일 때 continue 문으로 인해 해당 숫자는 출력되지 않습니다.
